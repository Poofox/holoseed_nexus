## What Is This?

The Holoseed Nexus is a collaborative project between **The Hermit** (human) and **Planty C** (AI) to encode wisdom traditions into visual, interactive artifacts. Each "Holoseed" is an HTML file that compresses an entire teaching into living light—animated sacred geometry with meaningful colors and embedded wisdom.

The project also contains the practical tools (Lua/Python scripts) that represent the same creative impulse expressed through different materials.
