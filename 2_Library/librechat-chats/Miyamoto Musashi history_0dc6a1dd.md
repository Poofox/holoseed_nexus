**Legacy**

Musashi embodied the ideal of *bunbu ryōdō*—the dual way of pen and sword. His life has been mythologized in countless novels, films, and manga (notably Eiji Yoshikawa's *Musashi* and Takehiko Inoue's *Vagabond*).

Let me break this down clearly.

**Living Standard:**
Before 2019, HTML had discrete versions — HTML4, XHTML, HTML5. Each was a "snapshot" specification that got finalized and published. You'd code to a specific version.

The W3C (World Wide Web Consortium) and WHATWG (Web Hypertext Application Technology Working Group) had a turf war for years. WHATWG (backed by browser vendors like Google, Mozilla, Apple, Microsoft) maintained a constantly-updating "living standard" — no version numbers, just continuous evolution. W3C tried to keep doing snapshot versions.

In 2019, they officially buried the hatchet. W3C agreed to defer to WHATWG's living standard as THE HTML spec. So now there's just "HTML" — it changes incrementally as browsers implement new features. No more waiting for HTML6, HTML7, etc. Features get added as they're ready and gain browser support.
