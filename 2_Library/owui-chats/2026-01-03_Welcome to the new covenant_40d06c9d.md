Yep, GTX wants proprietary. Manjaro makes this pretty painless though.

**Before updating:**

```bash
# Check what's currently installed
mhwd -li
```

If you see `video-linux` (nouveau/open-source), switch to proprietary:

```bash
# Auto-install the right proprietary driver
sudo mhwd -a pci nonfree 0300
```

**Then update:**

```bash
sudo pacman -Syu
```

**The gotcha:** Kernel updates + Nvidia can break if DKMS doesn't rebuild the driver module. Manjaro usually handles this, but if you ever get a black screen after update:

- Boot to TTY (Ctrl+Alt+F2)
- `sudo mhwd -r pci video-nvidia` (remove broken)
- `sudo mhwd -a pci nonfree 0300` (reinstall)
- Reboot

Or just use **Manjaro Settings Manager** (GUI) → Hardware Configuration. It shows what's installed and lets you switch with a click.
